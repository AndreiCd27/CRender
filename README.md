# CRender (3D Renderer in C++ realizat cu libraria OpenGL)

## Informatii generale

Codul este comentat destul de des, dar unele clase mai necesita niste redactare. In curand voi adauga niste comentarii mai "riguroase".
Comentariile din codul sursa au fost redactate in limba engleza pentru a putea fi intelese si de vitorii citiori.
